## 🎯 Objective

To analyze and summarize data using **aggregate functions**, `GROUP BY`, and `HAVING` clauses.
This lab focuses on extracting meaningful insights from relational data without modifying table structure.

## 🗂 Tables Used

* Student
* Enrollment
* Course
* Faculty
* Department

## 🧠 Concepts Covered

Aggregate Functions • GROUP BY • HAVING • COUNT() • MAX() • MIN() • Multi-table Aggregation • Filtering Groups

## 📌 Parts Covered

**Part A:** Basic Aggregation (COUNT, MAX, MIN)
**Part B:** GROUP BY Operations
**Part C:** HAVING Clause Filtering
**Part D:** Course-wise Analysis
**Part E:** Department-wise Analysis
**Part F:** Advanced Aggregation Queries

## ▶ How to Run

1. Create `DB-Lab-6.sql`
2. Paste all corrected queries
3. Run using SQL*Plus / Oracle SQL Developer / VS Code SQL Extension
